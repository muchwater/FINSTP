주간 뉴스 브리프 생성 프로그램

사용 설명서 (인수인계 자료) · KAIST 국가미래전략기술 정책연구소

이 프로그램이 하는 일

이번 주 뉴스 내용만 정해진 칸에 적어주면, PDF와 똑같은 모양의 이메일용 주간 뉴스 브리프를 자동으로 만들어 줍니다. 매주 이메일 편집기에서 레이아웃을 손으로 꾸밀 필요 없이, 내용만 바꿔 실행하면 완성된 화면이 브라우저에 뜹니다.

준비물

딱 하나만 있으면 됩니다 — 이번 주 내용이 적힌 파일

폴더 안에 있는 sample_data.json 파일이 예시이자 틀입니다. 이 파일을 메모장으로 열어 내용만 바꿔 쓰면 됩니다.

단계별 사용 방법

1파일 열기

sample_data.json 파일을 마우스 오른쪽 클릭 → 연결 프로그램 → 메모장으로 열어주세요.

2내용 바꿔 쓰기

파일 안쪽 모습은 아래처럼 생겼습니다. 따옴표("") 안에 적힌 글자만 바꿔주세요. 괄호나 쉼표, 기호는 건드리지 않습니다.

"period": "2026년 4월 1주차",              ← 주차
"date_range": "2026.04.01 ~ 2026.04.05",  ← 날짜 범위

"key_issues": [                           ← 이번 주 핵심 이슈 (3개)
  "첫 번째 이슈 내용",
  "두 번째 이슈 내용",
  "세 번째 이슈 내용"
],

"articles": [                             ← 기사 목록
  {
    "category": "반도체 디스플레이",        ← 분야
    "source": "전자신문",                  ← 신문사
    "date": "2026.03.22",                 ← 기사 날짜
    "title": "기사 제목을 여기에",
    "summary": "기사 요약 내용을 여기에",
    "url": "https://기사링크주소"
  },
  ...
]
주의사항
  • 한 기사가 끝나면 }, 를 꼭 붙입니다 (마지막 기사만 } 로 끝남)
  • 따옴표 안에서 엔터(줄바꿈)를 누르지 마세요
  • 큰따옴표 " 를 빠뜨리지 않도록 주의
3저장하기

Ctrl + S 를 눌러 저장한 뒤 메모장을 닫습니다.

4프로그램 실행하기

html_format 폴더를 열고, 폴더 안 빈 공간에서 Shift 키를 누른 채 마우스 오른쪽 클릭 하면 나오는 메뉴에서 다음 중 하나를 고릅니다:

  • "여기에서 PowerShell 창 열기"
  • "여기에서 터미널 열기"
  • "여기에 명령 창 열기"

검은 창이 뜨면 아래 한 줄을 복사해서 붙여넣고 엔터를 누릅니다.

python generate_brief.py sample_data.json --open

▶ 실행이 끝나면 자동으로 웹브라우저가 열리면서 완성된 결과물이 보여집니다.

5결과물 확인

브라우저 창에 주간 브리프가 나타나면 완성입니다.

수정하고 싶으면 다시 2번 단계 (메모장에서 내용 바꾸기) 로 돌아가 고친 뒤 4번을 다시 실행하시면 됩니다.

이메일로 보내는 법

방법 A — 파일 그대로 첨부하기

  1. output 폴더 안에 생긴 brief_2026_04_w1.html 파일을 이메일에 첨부합니다
  2. 받는 분이 첨부파일을 열면 브라우저에서 바로 볼 수 있습니다

방법 B — 이메일 본문에 넣기 (Gmail 추천)

  1. 브라우저에 열린 결과물 화면 위에서 Ctrl + A (전체 선택)
  2. Ctrl + C (복사)
  3. Gmail 새 편지 창에서 Ctrl + V (붙여넣기)
  4. 발송
Tip. 헤더 이미지와 KAIST 로고가 결과물 파일 안에 함께 포함돼 있어서, 받는 분에게도 그림이 깨지지 않고 그대로 보입니다.

문제가 생겼을 때

상황해결 방법
"python 을 찾을 수 없습니다" 오류 파이썬이 설치되어 있지 않은 상태입니다.
python.org 에서 설치 후 "Add Python to PATH" 옵션을 반드시 체크하세요.
검은 창에 빨간 오류 메시지가 뜸 sample_data.json 에서 따옴표·쉼표·괄호를 빠뜨렸을 가능성이 큽니다. 메모장에서 다시 열어 기호들을 확인하세요.
결과물에 이미지(헤더·로고)가 안 보임 images 폴더 안에 header background.jpg 와 FINSTP logo.png 두 파일이 그대로 있는지 확인하세요. 파일 이름도 똑같아야 합니다.
브라우저가 자동으로 안 열림 output 폴더에 들어가 brief_2026_04_w1.html 파일을 더블클릭하세요.
한글이 깨져 보임 메모장으로 sample_data.json 을 저장할 때 "다른 이름으로 저장" → 인코딩을 UTF-8 로 지정해서 저장하세요.

매주 할 일 요약

이 5단계만 반복하시면 됩니다.
  1. sample_data.json 파일 열기
  2. 주차 · 날짜 · 이슈 · 기사 내용 바꾸기
  3. 저장 (Ctrl + S)
  4. 명령창에서 python generate_brief.py sample_data.json --open 실행
  5. 브라우저에 뜨는 결과물 확인 → 이메일로 발송

파일/폴더역할
